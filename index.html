Blixt Wallet is an open-source Lightning Bitcoin Wallet for Android with focus on usability and user experience, powered by lnd and Neutrino SPV.

It is currently aimed towards Bitcoiners who want to try out using Lightning Network.

View the github project.

Built with lnd and Neutrino SPV

Blixt Wallet uses the Lightning Network client lnd and the Bitcoin SPV client Neutrino under the hood, directly on the phone, respecting your privacy.
Information about your wallet does not leave your phone.

Simple and clean interface

The design philosophy behind Blixt Wallet is to provide a clean and straightforward interface for doing transactions.

Effort has been made to make sure that the transaction log is descriptive and clear.

Easy to get started

When you send on-chain funds to Blixt Wallet, it will automatically open channel(s) on your behalf, so you can get started in no time.

Feature-rich and cutting-edge

Blixt Wallet keeps up with new developments in the Lightning community, supporting features such as LNURL and WebLN as well as keeping up-to-date with the new features in the lnd project.

Open-source project

Blixt Wallet is built as an MIT-licensed open-source project.
