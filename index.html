Description

Blixt Wallet is a non-custodial open-source Bitcoin Lightning Wallet for Android and iOS with a focus on usability and powerful user experience.

It's currently aimed towards Bitcoiners who want to try out using Lightning Network, with a large variety of features available. Soon will be available the desktop version for Mac, Linux and Windows.

View the github project.

Feature-rich and cutting-edge technology

Key aspects to mention:

Built with LND and Neutrino

Blixt Wallet uses the Lightning Network client lnd and the Bitcoin SPV client Neutrino under the hood, directly on the phone, respecting your privacy.

NON-custodial

Blixt wallet does not use any centralized servers for doing transactions. User have full control over his private keys and broadcasting transactions.

Open-source project

Blixt Wallet is built as an MIT-licensed open-source project.

Innovation

Blixt Wallet keeps up with new developments in the Lightning community, supporting features such as Keysend, Lightning Address, LNURL and WebLN as well as keeping up to date with new features in the lnd project.

Simple and clean interface

The design philosophy behind Blixt Wallet is to provide a clean and straightforward interface and user experience for doing transactions. Effort has been made to make sure that the transaction log is descriptive and clear.

Easy to get started

Blixt Wallet offers multiple ways to get started with Lightning Network: auto-pilot, Dunder LSP and normal opening LN channels (see more details in features page and in guides page.
