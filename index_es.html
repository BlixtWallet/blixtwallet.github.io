Descripción

Blixt Wallet es un monedero Bitcoin Lightning de código abierto, sin custodia, para Android e iOS con un enfoque en la usabilidad y una experiencia de usuario poderosa.

Actualmente está dirigido a Bitcoiners que quieren probar el uso de Lightning Network, con una gran variedad de funciones disponibles. Próximamente estará disponible la versión de escritorio para Mac, Linux y Windows.

Ver el proyecto github.

Aspectos clave a mencionar:

Construido con LND y Neutrino

Blixt Wallet usa el cliente Lightning Network lnd Y el cliente de Bitcoin SPV Neutrino debajo del capó, directamente en el teléfono, respetando su privacidad.

NON-custodial

El monedero Blixt no utiliza ningún servidor centralizado para realizar transacciones. El usuario tiene control total sobre sus claves privadas y transacciones de transmisión.

Proyecto de código abierto

Blixt Wallet está construido como un licencia MIT - proyecto de código abierto.

Innovación

Blixt Wallet se mantiene al día con los nuevos desarrollos en la comunidad Lightning y admite funciones como Keysend, Lightning Address, LNURL y WebLN así como mantenerse al día con las novedades del proyecto lnd.

Interfaz simple y limpia

La filosofía de diseño detrás de Blixt Wallet es proporcionar una interfaz limpia y sencilla y una experiencia de usuario para realizar transacciones. Se ha hecho un esfuerzo para asegurarse de que el registro de transacciones sea descriptivo y claro.

Fácil de empezar

Blixt Wallet ofrece múltiples formas de comenzar con Lightning Network: piloto automático, Dunder LSP y canales LN de apertura normal (ver más detalles en pagina de caracteristicas y en página de guías).
